Задачи 4го дня.

Задача 1
    Дано целое положительное число n, (n<=100 000) затем n целых чисел,
    помещающихся в int. Определить сколько среди них есть различных чисел.

Задача 2
    В первой строке дано n - количество пользователей (0<n<=100 0), 
    В следующих n строках перечисленная информация в каких группах социальной
    сети пользователь состоит.
    В i-ой строке указаны через пробел id групп (целые положительные числа не больше 10^9)
    в которых находится пользователь с айдишником i. (нумерация идет с 1). 
    Нужно для каждой из групп выдать в отдельной строке:
    id группы, количество пользователей в группе,
    далее перечислить id пользователей, которые в нее входят. Все числа перечислить через пробел.
    Число различных групп не боле 1000.

Задача 3
    Дано n и n запросов вида:
    1 х - добавить шар с числом х в мешок.
    2 х - убрать ОДИН шар с числом х из мешка.
    В конце выдать сколько есть уникальных чисел на шарах в мешке.
    n,х<=10^5
